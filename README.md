# WeChatRobotCSharp
C# 版 微信PC WeChatRobot   
使用 C# 实现注入 WeChatHelper.dll , 通过 `WM_COPYDATA` 消息跟 WeChatHelper.dll 进行通讯。  
只实现了基础功能, 需要别的功能请自行Fork实现, 这里只是作为一个C#的Demo, 上图。
